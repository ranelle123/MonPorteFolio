
# MonPorteFolio

Portfolio statique simple construit avec HTML, CSS et JavaScript.

## Description
Ce dépôt contient une maquette de site personnel légère et responsive. Le contenu est organisé en pages HTML, avec des styles CSS dédiés et des interactions JavaScript simples.

## Structure
- `Projet.html` : page principale
- `Ranelle.html` : page alternative
- `Ranelle.css` : feuille de styles
- `Ranelle.js` : scripts JavaScript
- `images/` : ressources visuelles

## Utilisation
Ouvrez directement l’une des pages HTML dans un navigateur pour voir le site en action. Aucune installation n’est nécessaire.
