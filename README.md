
# MonPorteFolio

Portfolio statique en HTML/CSS/JS.

## Description
Site personnel responsive avec pages HTML, styles CSS et interactions JS.

## Fichiers
- `Projet.html` : page principale
- `Ranelle.html` : page alternative
- `Ranelle.css` : styles
- `Ranelle.js` : scripts
- `images/` : images

## Usage
Ouvrir une page HTML dans le navigateur.
